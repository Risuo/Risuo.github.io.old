GPU built Google Quick-Sketch CNN

This page leans heavily on the Sketcher project develoepd by Zaid Alyafeai. His page is available here: Sketcher. The CNN on this page was trained to recognize all 345 classes using the quick draw dataset.

This project is an extension of Zaid's implementation. Rather than limiting to 100 classes, I wanted to see how it would perform under all 345 classes.

Unsurprisingly, the accuracy is lower, however with training the model was able to achieve almost 90% Top 5 accuracy.
